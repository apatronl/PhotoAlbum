# PhotoAlbum
This is a project I built for CS 4470: User Interface Software class at Georgia Tech.

# Features
- Gestures
- Tagging
- Annotations

# How to Run
Once you have downloaded the album.jar file, run the following command:

```sh
java -jar album.jar
```
