To enable magnet mode make sure “magnet mode” is selected while in grid view.

Extra Credit Implemented
- In magnet mode, you can select a thumbnail and double-click on it to switch to photo view. Thumbnails can also be tagged/untagged while in magnet mode.
- Zooming functionality also works in magnet mode, so thumbnails can be scaled up or down.
